# USA House Price Regression

This project predicts house prices in U.S. states and territories using regression techniques from statistical learning.

## Main Notebook

- `notebooks/USA_house_price_regression.ipynb`

## Data

Included:

- `data/USA Real Estate Dataset_sample.csv`

Optional full dataset:

- `data/USA Real Estate Dataset.csv`

The notebook automatically uses the full dataset if it is present in the `data/` folder. Otherwise, it runs with the sample file included in this repository.

## Main Topics Covered

- Multiple linear regression
- Interaction terms
- Polynomial regression
- Forward and backward stepwise selection
- Ridge and lasso
- Step functions
- Regression splines
- Smoothing splines
- Local regression
- KNN regression
- Five-fold cross-validation
- Validation-set comparison and final ranking

## Practical Note

The original real-estate dataset is too large for a standard GitHub upload, so this repository includes a sample for portability and demonstration.
