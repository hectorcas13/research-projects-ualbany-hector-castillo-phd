# Fraud Classification

This project analyzes a fraud-detection problem using a banking transaction dataset and statistical learning methods for classification.

## Main Notebook

- `notebooks/Fraud_banking_classification_stratified_80_20.ipynb`

## Dataset

- `data/fraud_data.csv`

## Main Topics Covered

- Data cleaning and feature engineering
- Binary target setup
- Logistic regression
- Forward and backward stepwise selection
- LDA and QDA
- Naive Bayes
- Polynomial logistic regression
- KNN classification
- Ridge and lasso
- Five-fold cross-validation
- Training/testing comparison and model ranking

## Sharing Note

The notebook in this folder uses relative paths, so it can be opened from this repository without depending on machine-specific absolute paths.
